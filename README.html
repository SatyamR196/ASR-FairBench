Run Locally

Prerequisites :

  • Node.js: Make sure you have Node.js v22.12.0 or higher installed on your machine. You can download it from Node.js official website.
  • Google Colab: You will need a Google account to access Google Colab.
  • Ngrok: Sign up for a free account on ngrok to expose your local server to the internet.

Frontend :

Clone the project from GitHub to your local machine.

  git clone https://github.com/SatyamR196/ASR-FairBench.git

Go to the project directory

  cd ASR-FairBench

Install dependencies

  npm install

Start the react app

  npm run dev
